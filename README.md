vendor-icons
============


About
--------

Vendor icons (from HTML5/CSS3 and JavaScript universes)


This project just provides icons from various projects. All icons are made available through the following dimensions (in pixels):

* 16x16
* 32x32
* 48x48
* 64x64
* 96x96
* 128x128
* 256x256
